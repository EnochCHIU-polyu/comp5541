# Transfer Learning

Transfer learning involves taking a model pre-trained on a large dataset (e.g., ImageNet) and adapting it to a new, often smaller, dataset. Key steps include freezing early layers, replacing the output layer, and fine-tuning with a lower learning rate. This leverages generic features learned from the source task.
