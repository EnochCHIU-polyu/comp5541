# Convolutional Neural Network (CNN)

A Convolutional Neural Network is a class of deep neural networks commonly used for analyzing visual imagery. It consists of convolution layers, pooling layers, and fully-connected layers. CNNs are designed to automatically and adaptively learn spatial hierarchies of features from input images.
