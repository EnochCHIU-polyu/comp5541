# ReLU Activation Function

ReLU (Rectified Linear Unit) is a popular activation function defined as f(x) = max(0, x). It helps mitigate the vanishing gradient problem by providing a non-saturating gradient for positive inputs. ReLU is computationally efficient and promotes sparse activations, aiding optimization.
