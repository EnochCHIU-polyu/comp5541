# Sigmoid Activation Function

The Sigmoid function squashes input values into the range [0,1], historically used to interpret outputs as probabilities or biological neuron firing rates. However, it suffers from gradient saturation, where large magnitude inputs lead to near-zero gradients, causing the vanishing gradient problem in deep networks.
