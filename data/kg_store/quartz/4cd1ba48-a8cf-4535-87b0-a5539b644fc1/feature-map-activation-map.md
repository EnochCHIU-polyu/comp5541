# Feature Map (Activation Map)

A feature map, or activation map, is the output of applying a convolutional filter to an input image. Each feature map highlights the presence of specific learned features (e.g., edges, textures) at different spatial locations. Shallow layers tend to produce maps representing generic local features, while deeper layers capture more specific, abstract patterns.
