# ReLU Activation

ReLU (Rectified Linear Unit) is a widely used activation function defined as max(0,x). It is computationally efficient and helps alleviate the vanishing gradient problem by not saturating for positive inputs. ReLU promotes sparse activation, which can aid in optimization, though it may suffer from 'dying neurons' where gradients become zero for negative inputs.
