# Convolution Layer

A convolution layer applies a set of learnable filters to an input image to produce an activation map or feature map. Each filter computes a dot product between its weights and a small region of the input, effectively detecting local features like edges and textures. The spatial dimensions of the output are determined by parameters like stride and padding.
