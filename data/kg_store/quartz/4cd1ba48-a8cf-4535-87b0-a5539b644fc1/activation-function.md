# Activation Function

An activation function introduces non-linearity into a neural network, allowing it to learn complex patterns. It affects gradient flow during backpropagation, impacting optimization and training dynamics. Common choices include Sigmoid, tanh, ReLU, and Leaky ReLU, each with different properties regarding representation power, computational cost, and ease of optimization.
